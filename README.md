# Agentic_platform
## Summary
Проект по разработке внутренней агентской платформы компании. Планируемая система должна покрывать все 
текущие разрозненные ML кейсы компании единым высокопроизводительным ядром с гибкой настройкой сценариев 
с использованием агентов и без них.

Выпускной проект курса ML System Design ITMO.

## Installation
Должен быть установлен Python >=3.12. Далее установка через пакетный менеджер UV:

```commandline
pip install uv
```

```commandline
uv sync
```

## Running the project
Будет дополнено позднее.

Работа с созданным venv обеспечивается с запуском по-умолчанию:

```commandline
.venv/Scripts/Activate
```

или 

```commandline
uv run
```
